Portfólio

Este é o meu primeiro projeto de página web desenvolvido durante meus estudos de Front-End, marcando o início da minha trajetória prática no desenvolvimento web. Mais do que apenas criar uma página, este projeto representa minha dedicação em aprender como transformar ideias em interfaces funcionais, organizadas e visualmente agradáveis. Durante o desenvolvimento, pude colocar em prática conceitos fundamentais de HTML, CSS e estruturação de páginas, além de desenvolver meu raciocínio lógico, criatividade e atenção aos detalhes. Como estudante de Tecnologia e futura desenvolvedora, estou construindo cada projeto como um passo importante na minha evolução profissional. Tenho grande interesse em criar soluções úteis, intuitivas e que proporcionem uma boa experiência para o usuário. Este projeto simboliza o começo de uma jornada que está sendo construída com muito esforço, curiosidade e paixão pela tecnologia. E esse é apenas o primeiro de muitos desafios e projetos que ainda virão.
